# Supply-Chain-Logistics-RAG Configuration

## Project Overview

A Python 3.14 localized RAG pipeline for tactical supply chain metrics.

## Environment & Package Manager

- This project uses `uv` for python environment isolation and dependency tracking.
- Do not use standard `pip` or `venv` directly.

## Execution Commands

- To run data ingestion: `uv run src/ingestion.py`
- To run the RAG pipeline: `uv run src/rag_pipeline.py`
